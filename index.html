    Fitfolio

    Fitfolio is a digital closet app that helps users photograph, organize, and plan outfits using their real wardrobe. We wanted to tackle outfit repetition, unworn clothing, and the time it takes to plan outfits. Our solution digitizes users' wardrobe so they can mix and match items and use the Fashn API to visualize outfits on their avatar.

    NoteApp

    A note-taking app to help user's prioritize and organize their tasks.

    click to expand

    NoteApp

    NoteApp is a note-taking and to-do list app. We added features such as user authentication, allowing users to create tasks, set priorities, sort by importance, and receive push notification reminders to help users stay organized.

    Mirror Track

    A 3D interactive floating track made with Three.js.

    click to expand

    Mirror Track

    A 3D interactive scene built with Three.js featuring two cars traveling along a figure-8 track suspended in a skybox environment. The scene depicts reflective chrome spheres that mirror the surrounding skybox, creating the illusion of a floating track in space.
